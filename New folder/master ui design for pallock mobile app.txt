Master UI/UX Design Prompt: "Pal-Luck" Mobile App
Role & Objective:
Act as an expert Mobile UI/UX Designer. Create a high-fidelity, interactive mobile app interface for an application named "Pal-Luck". The app is a hybrid ecosystem blending communication, community discovery, and a marketplace.

Global Design Philosophy & Vibe:

Aesthetic: High-end iOS/iPhone feel. The design must feature deep spatial depth, subtle glassmorphism (frosted glass effects), smooth modal transitions, and dynamic parallax interactions.

Color & Lighting: The overall theme is anchored by a luminous, trusting "blue glow." Use deep OLED blacks or pristine whites as a canvas, accentuated by soft, breathing blue glowing elements (shadows, active buttons, and highlights) to make the interface feel alive and premium.

Layout Rule: Rely on the provided visual references (Instagram-style modals, Google-style pristine search, WhatsApp-style minimal chats).

Phase 1: The Onboarding Flow (First Impressions)
Screen 0.1: Landing / Splash Screen (Reference: iPhone Videoframe)

Visuals: A visually striking parallax experience. The background features a subtle, dynamic sunset/deep blue gradient that shifts as the phone moves.

Centerpiece: The Pal-Luck logo (two arms holding each other, symbolizing trust) sits dead center. It must have a premium "iPhone feel"—a sleek, 3D-like, soft-glowing presence that seems to float slightly above the background.

Interaction: The logo exhibits a gentle, breathing blue glow. Tap anywhere or swipe up to smoothly slide into the next screen.

Screen 0.2: Phone Number Entry

Visuals: The background seamlessly transitions to a clean, minimal interface (light or dark mode, but maintaining the subtle blue glowing accents on active fields).

Elements: A large, elegant typography prompt asking "Let's get started." Below it, a clean input field for the phone number with a built-in country code selector.

Action: A prominent, glowing blue "Continue" button at the bottom.

Screen 0.3: Verification (OTP)

Visuals: Maintains the exact background of the previous screen to avoid jarring context switches.

Elements: "Enter your verification code." Below are 4 to 6 large, clean, rounded-square input boxes. As the user types, the active box lights up with a soft blue outer glow.

Details: Include a subtle "Resend code in 0:30" text below, using a muted gray tone.

Screen 0.4: Profile Setup

Visuals: A warm, inviting setup page.

Elements: A large, circular placeholder for a profile picture in the upper-center. It features a glowing blue + or camera icon indicating it's clickable to upload. Below the picture, a sleek text input line for "Your Name".

Action: A "Complete Setup" button that completes the onboarding with a satisfying, fluid checkmark animation, launching the user into the main app.

Phase 2: Main Application Features
Global Navigation Bar (Bottom):
A floating, frosted-glass (iOS style) bottom tab bar with 6 perfectly spaced icons: [ Chat ], [ Community ], [ Pal-Lock ] (elevated and glowing), [ Dock ], [ Status ], [ Settings ].

1. Chat Section (Reference: Chat Image)

Vibe: WhatsApp simplicity but with iOS elegance.

Layout: A clean, vertical list of active conversations. Profile pictures on the left, bold names, and preview text. Include a soft blue notification dot for unread messages.

Inside Chat: Clean, rounded message bubbles. Your messages are highlighted in the signature Pal-Luck glowing blue.

2. Community Section (Reference: Community & Instagram Images)

Vibe: Instagram's visual fluidity meets structured community boards.

Layout: A visual feed with zero full-page reloads. Tapping on a community post, store, or profile slides up a smooth, frosted-glass bottom sheet modal.

Top Bar: A horizontal, scrollable row of pill-shaped "Smart Filters" (Communities, Stores, Groups, Locations). Active filters glow slightly.

3. Pal-Lock Section (Reference: Google First & Pal-Lock Images)

Vibe: The core discovery engine. Google-level focus mixed with an intelligent grid.

Search Interface: A highly polished, centered search bar that dominates the upper screen. It should look inviting, featuring a camera icon 📷 for visual searches. Underneath, subtle, fading text suggests: "Upload a picture to find items..."

Results Display: Transitions into an immersive, Pinterest-style Masonry Grid. Cards display high-quality images of items/services, accompanied by a trust-level badge. Each card features a clear "Lock-in" action button to initiate instant connection.

4. Dock Section (Reference: Dock Dashboard Feel)

Vibe: A personal command center.

Layout: Card-based folders organizing My Items, My Services, and Saved Items. Use soft shadows to give the cards a physical, layered appearance against the background.

5. Status & Settings

Status: A full-screen, vertical scrolling interface (like Instagram Reels/TikTok) optimized for rich media, with UI elements (likes, profile, share) floating cleanly over the video/image.

Settings: iOS-native feeling. Simple, text-based, highly organized list with clear dividers, chevron arrows > for navigation, and toggles that feature the signature blue color when activated.

Developer / Designer Note: Every interaction should feel weightless. Use skeleton loaders instead of spinning wheels, heavy reliance on blur/glassmorphism for overlays, and ensure the "blue glow" is used selectively to guide the user's eye, not overwhelm it.
