Master UI/UX Design Prompt: "Pal-Luck" Mobile AppRole & Objective:Act as an expert UI/UX Designer. Your task is to design a high-fidelity mobile app interface for an application named "Pal-Luck". The app combines the communication utility of a messenger with the visual discovery of a marketplace.Global Design Philosophy & Vibe:Core Principle: The user should never feel lost. The interface must feel fast, familiar, intelligent, and trust-driven.Aesthetic: Modern, calm, friendly, and trustworthy.Color Palette: Clean white backgrounds, soft blue accents (to signify trust), and light, smooth gradients.UX Hybrid Model: * WhatsApp for simplicity (no confusion).Instagram for fluidity (smooth modal pop-ups, no full page reloads).Google for smart, clean search.Pinterest for visual result displays.Global Navigation:Bottom Tab Bar: Implement a sticky bottom navigation bar with 6 distinct sections: [ Chat ], [ Community ], [ Pal-Lock ], [ Dock ], [ Status ], [ Settings ].The center tab ([ Pal-Lock ]) should be slightly highlighted or elevated.Screen-by-Screen Requirements:1. Splash / Welcome Screen & OnboardingVisuals: A soft sunset gradient background (referencing a smooth glowing animation).Centerpiece: The Pal-Luck logo with a subtle breathing/glowing effect, accompanied by a minimalist graphic of two arms holding each other (symbolizing trust).Onboarding: A stark contrast to the splash screen—clean, minimal white background with soft blue accents. Features a simple phone number input, country selector, and a "Continue" button for frictionless signup.2. Chat SectionReference: WhatsApp.Layout: A standard chat list showing profile pictures, user names, and the last message snippet.Elements: Include a floating "New Chat" button. Inside the chat, use clean message bubbles with minimal distractions. The UI should imply zero learning curve.3. Community SectionReference: Instagram (visual feed) + Twitter (threaded discussions).Top Bar: A "Smart Filters Bar" featuring a search input and pill-shaped toggle filters for: Communities, Stores, Groups, Locations, and adjustable scopes (Nearby, Global).Feed Layout: Scrollable content featuring posts, stores, and community highlights. Include clean, threaded reply structures for discussions.Interaction: Clicking a profile or store should open as a smooth bottom-sheet overlay/modal, with a clear "Exit/Cancel" button at the top.4. Pal-Lock Section (The Core Feature)Reference: Google (search input) + Pinterest (results).Search Interface: A pristine white screen dominated by a centered, prominent search bar. Include a camera icon (📷) inside the search bar for image uploads.Empty State Text: Include subtle, intelligent AI suggestion text below the bar, such as "Looking for a mechanic?" or "Upload a picture to find items".Results Interface: Once a search is executed, transition into a dark or light mode Masonry Grid (like Pinterest). Each card should display an item/service image, the user/store name, and a "trust level" indicator. Include a prominent "Lock-in With a Pal" call-to-action button to instantly start a chat.5. Dock SectionVibe: A personal dashboard or mini-storefront.Navigation: Top or sub-tabs for: My Items, My Services, My Store, and Saved Items.Layout: A neat folder system to organize items into categories and collections. Include clear action buttons to "Upload product" or "Add service".6. Status SectionReference: TikTok / Instagram Reels.Layout: A full-screen, vertical scrolling interface dedicated to passive discovery. Show videos, images, and service highlights with transparent overlay text and profile links.7. Settings SectionReference: iOS Settings / Instagram Settings.Layout: Extremely clean, list-based, text-heavy UI.Top: Search bar for settings.Categories: Use clear dividers with distinct sections for Profile (Name, Photo), Search Preferences (Location radius), Privacy & Security (2FA, Blocklist), Chat Settings (Themes), and Account. Use minimalist chevron icons (>) to indicate clickable rows.
